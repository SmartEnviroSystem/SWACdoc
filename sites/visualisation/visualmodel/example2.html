Example Box styleing

This example shows how to set box styleing useing dataset attributes.

VIEW-Requestor
CONFIGURATION

There is no configuration needed for this example.

DATASOURCE
Attribute Description
name Name the visual element
desc Visualisations description
width Width of the box
height Height of the box
elem_margin Inner margin of elem.
elem_fillcolor Color that is the visualisation filled with.
elem_bordercolor Color of the visualisations border.
elem_kind
name description
rect Default visualisation. Dimension attributes: width & height
circle Circle visualisation. Dimension attributs: radius
ellipse Ellipse visualisation. Dimension attributes: radiusX & radiusY
wedge Wedge visualisation. Dimension attributes: radius & angle
star Star visualisation. Dimension attributes: innerRadius & outerRadius & numPoints
ring Ring visualisation. Dimension attributes: innerRadius & outerRadius
arc Arc visualisation. Dimension attributes: innerRadius & outerRadius & angle
regularpolygon Visualisation as regular polygon. Dimension attributes: radius & sides
