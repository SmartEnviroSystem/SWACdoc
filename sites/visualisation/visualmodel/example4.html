Example 4: Label styleing

The labels near an element can be styled in font and color.

Notice: This options need to be bugfixed.
VIEW-Requestor
CONFIGURATION

There is no configuration needed for this example.

DATASOURCE
Attribute Description
labelFrontFamily Labels Font Family (e.g. Arial, Times New Roman, ...)
labelFrontStyle Labels Font Style
labelFrontSize Labels text size
labelStrokeWidth Stroke width for the label.
labelFillColor Filling color for label
labelBorderColor Border color for label
labelPadding Label text padding from label border
labelX Relative x position to the element the label belongs to.
labelY Relative y position to the element the label belongs to.
